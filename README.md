#  Rey de Copa - Sistema de Gestão de Vendas

Este é um sistema de gerenciamento de vendas desenvolvido em **C++**, focado na organização e persistência de dados para um e-commerce. O projeto foi criado como parte dos meus estudos no 4º período de **Ciência da Computação**, aplicando conceitos de estruturas de dados e manipulação de arquivos.

##  Funcionalidades

O sistema permite gerenciar o fluxo de produtos da loja "Rey de Copa" através de uma interface de linha de comando (CLI):

* **Registro de Vendas**: Cadastro de produtos com nome, preço e quantidade.
* **Persistência de Dados**: Armazenamento automático em arquivo `.txt`, garantindo que os dados não sejam perdidos ao fechar o programa.
* **Visualização de Histórico**: Listagem numerada de todos os registros salvos para conferência.
* **Arquitetura Modular**: Código organizado em funções e utilizando `structs` para melhor manutenção.

## Tecnologias Utilizadas

* **Linguagem**: C++.
* **Bibliotecas Standard**: `iostream`, `fstream`, `string`, `vector` e `cstdio`.
* **Sistema Operacional**: Desenvolvido e testado em **Pop!_OS (Linux)**.
